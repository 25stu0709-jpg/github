## 3. README.md 업로드 방법 (GitHub에 올리기)

1. GitHub에서 내 저장소에 들어가요.

2. **"Add file"** 버튼 클릭 후 **"Upload files"** 선택해요.

3. 방금 만든 `README.md` 파일을 선택해 올려요.

4. **Commit changes** 버튼 클릭!

---

## 4. 그럼 어떻게 보여요?
